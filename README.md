# GPT 图片生成工具（Python + Vue3 本地版）
开源免费的本地AI绘画聚合工具，兼容各类主流AI绘画API，所有数据本地存储，隐私安全，无需担心数据泄露。

## ✨ 核心优势
- 🚀 **开箱即用**：支持Docker一键启动，无需复杂配置
- 🔒 **隐私安全**：所有任务记录、生成图片、配置信息全部存储在本地浏览器，后端仅做请求转发，不存储任何数据
- 🎯 **多API兼容**：支持OpenAI DALL-E、Midjourney代理、Stable Diffusion、通义万相等各类AI绘画接口
- 🌐 **无跨域问题**：本地部署完美解决浏览器跨域限制，访问速度快
- 💾 **数据自主可控**：支持任务批量导入导出，数据迁移备份方便
- 🎨 **功能完整**：覆盖文生图、图生图、局部重绘等所有主流AI绘画功能
- 🔄 **高可用保障**：内置请求重试、多密钥自动轮询机制，大幅提升生成成功率
- 📦 **高效存储**：支持生成后自动压缩图片，兼顾画质和下载速度，节省存储空间

## 技术栈
### 前端
- Vue 3.4 + TypeScript
- Pinia 状态管理
- Element Plus UI 组件库
- Tailwind CSS
- Vite 构建工具

### 后端
- FastAPI
- Uvicorn ASGI 服务器
- Pydantic 数据验证
- Requests 请求转发

## 快速开始
### 1. 启动后端
```bash
cd backend
# 建议使用虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖（升级版本需要重新执行）
pip3 install -r requirements.txt

# 启动服务
python3 -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```
后端接口文档：http://127.0.0.1:8000/docs

### 2. 启动前端
```bash
cd frontend
npm install
npm run dev
```
前端访问地址：http://localhost:5174

### 3. Docker 一键启动
```bash
docker-compose up -d
```
访问：http://localhost:5174

## 🎯 功能特性
### 核心生成能力
- ✅ **文生图**：支持自定义提示词、负提示词、图片尺寸、生成数量、迭代步数、CFG系数、随机种子等完整参数配置
- ✅ **图生图/参考图编辑**：支持上传参考图，调整参考图权重、重绘强度，实现风格迁移、内容修改等效果
- ✅ **局部重绘**：支持自定义蒙版区域，只修改图片指定部分，保留其他区域内容不变

### 配置管理
- ✅ **多API兼容**：支持自定义各类AI绘画接口的基础地址、API密钥、接口路径，兼容OpenAI、Stable Diffusion等主流API规范
- ✅ **默认配置预置**：后端支持配置默认API地址和密钥，首次使用无需重复填写
- ✅ **前端可视化配置**：所有参数都可以在前端设置页面调整，无需修改代码
- ✅ **多API密钥轮询**：支持配置多个API密钥，自动轮询使用，遇到密钥失效自动切换，避免单密钥限流/额度用尽问题

### 数据管理
- ✅ **本地持久化存储**：所有任务记录、生成的图片、用户配置全部存储在浏览器IndexedDB中，断网也可查看历史记录
- ✅ **任务历史管理**：支持查看所有历史生成任务，按时间/关键词筛选搜索，一键重新生成、复制参数、删除单个/批量删除任务
- ✅ **数据导入导出**：支持批量导出所有任务（包含图片）为ZIP备份文件，也可导入备份文件恢复数据，跨设备迁移方便
- ✅ **批量打包下载**：支持多选任务一键打包所有图片为ZIP下载，无需逐个保存，批量导出更高效

### 使用体验
- ✅ **响应式界面**：支持桌面端、移动端自适应，不同设备都有良好使用体验
- ✅ **智能请求重试**：自动对网络超时、5xx服务端错误、429限流错误进行指数退避重试，前端无感知提升生成成功率
- ✅ **可选图片压缩**：支持开启生成后自动压缩图片，自动适配PNG/JPG格式，保留原画同时提供压缩版下载，兼顾画质和加载速度
- ✅ **双版本下载**：每张生成图片提供「下载原画」和「下载压缩图」两个选项，满足不同场景需求
- ✅ **图片批量操作**：支持单张/批量下载图片，复制图片到剪贴板
- ✅ **无跨域问题**：本地后端转发请求，完美解决浏览器跨域限制，无需依赖第三方代理服务

## 配置说明
所有配置都可以在前端设置页面填写，不需要修改后端代码，后端仅做请求转发，不存储任何数据。

### 后端环境配置
复制 `.env.example` 为 `.env` 可自定义后端配置：
```bash
cd backend
cp .env.example .env
```
支持配置：
- `DEFAULT_API_BASE_URL`：默认API基础地址
- `DEFAULT_API_KEY`：默认API密钥
- `REQUEST_TIMEOUT`：请求超时时间（秒）

## 常见问题
### 1. 后端启动报错 `uvicorn: command not found`
使用完整命令启动：
```bash
python3 -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

### 2. 后端启动报错 `SyntaxError: non-default argument follows default argument`
代码已修复，直接拉取最新版本即可正常运行。

### 3. 前端页面空白
- 确保后端服务已正常启动在 8000 端口
- 按 `Cmd+Shift+R` 强制刷新浏览器清除缓存
- 确认访问地址为 http://localhost:5174

### 4. 接口请求404
后端路由已统一为 `/api` 前缀，无需额外配置，前端代理已默认设置。

### 5. 多API密钥怎么配置？
在设置页的API密钥输入框中每行填写一个密钥，系统会自动轮询使用，遇到401/403错误会自动标记失效并切换到下一个密钥。

### 6. 图片压缩功能如何开启？
在设置页开启「生成后自动压缩图片」开关即可，生成时会自动生成压缩版图片，每张图片同时提供原画和压缩图两个下载选项。
